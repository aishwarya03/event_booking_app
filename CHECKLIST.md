# 🚀 Hackathon Execution Checklist

## 1. Understand the Problem
- [ ] Read the problem statement completely
- [ ] Clarify doubts with mentors/judges
- [ ] List mandatory features
- [ ] List bonus features
- [ ] Decide MVP

---

## 2. Database Design
- [ ] Define entities
- [ ] Draw ER diagram
- [ ] Create Prisma models
- [ ] Run Prisma migration

---

## 3. Backend
- [ ] Create routes
- [ ] Create controllers
- [ ] Create services
- [ ] Add validations (Zod)
- [ ] Connect Prisma
- [ ] Test APIs

---

## 4. Frontend
- [ ] Create pages
- [ ] Build reusable UI
- [ ] Connect APIs
- [ ] Validate forms
- [ ] Handle loading states
- [ ] Handle error states

---

## 5. Final Testing
- [ ] Test all user flows
- [ ] Fix major bugs
- [ ] Remove console.log()
- [ ] Check responsiveness
- [ ] Verify environment variables

---

## 6. Deployment
- [ ] Deploy backend
- [ ] Deploy frontend
- [ ] Test deployed application
- [ ] Update README (if required)

---

## 7. Before Submission
- [ ] Push latest code
- [ ] Verify teammates' commits
- [ ] Confirm deployment link works
- [ ] Confirm repository is accessible
- [ ] Submit before deadline
